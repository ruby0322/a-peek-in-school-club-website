## Tech Stack

- Prototype
    - Canva
- Frontend
    - Plain HTML & CSS
    - jQuery for RWD
    - no CSS Library
    - Google Fonts
- Backend
    - Python Flask
    - Python gunicorn
- Server
    - Deployed to Heroku
